Mario Hotaj

# Compilatation
java *.java

# Lancement
java Serveur -> pour initialiser le serveur
java Fenetre -> pour lancer une fenêtre (à lancer autant de fois que l'on veux)

# Ajout supplémentaire :
- Notification quand on essaye de se connecter avec un nom déjà pris. (Pop-up msg d'erreur)

- Fermeture de la fenêtre = notification de déconnexion au serveur

- Envoie de message privée via la commande "/mp personne message".
 (+) Si la personne n'existe pas on notifie l'expéditeur du message que le correspondant
N'existe pas.

- Code couleur pour chaque différents évènement dans le chat

- Possibilité d'envoyer des smileys en insérant :nomdusmiley: dans le message
 (+) On pourra ajouter des smileys au format .gif dans le dossier emoji/, son appel se fera par :{le nom du nouveau fichier sans l'extension .gif}:
